# Shibuya Record

**Shibuya Record** is Simple O/R Mapper on Ruby. Like **Active Record** and more simple **and Lesser.**

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'shibuya_record'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install shibuya_record
